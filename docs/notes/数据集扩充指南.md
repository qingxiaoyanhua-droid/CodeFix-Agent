### 方案 2: 爬取公开数据集（500-1000 条）

#### 2.1 QuixBugs（推荐）

QuixBugs 是一个经典的 bug 修复数据集，包含 40 个 Python bug。

```python
from datasets import load_dataset

# 加载 QuixBugs
data = load_dataset('Jeongmin/QuixBugs', split='train')

# 转换为 DPO 格式
dpo_data = []
for item in data:
    dpo_data.append({
        "prompt": f"""Bug: Fix the bug

Buggy Code:
```python
{item['buggy_code']}
```

Fix the bug.""",
        "chosen": f"""<reasoning>
{item['buggy_code']} has a bug in the logic.
</reasoning>
<fixed_code>
{item['fixed_code']}
</fixed_code>""",
        "rejected": f"""<reasoning>
Not sure about this fix.
</reasoning>
<fixed_code>
{item['buggy_code']}
</fixed_code>"""
    })
```

## ⚠️ 注意事项

### 1. 数据质量
- 确保 chosen 是正确的修复
- 确保 rejected 是有问题但不完全错误的（增加难度）
- 避免完全重复的数据

### 2. 数据格式
DPO 需要特定格式：
```json
{
    "prompt": "Bug: Fix...",
    "chosen": "<reasoning>...</reasoning><fixed_code>...</fixed_code>",
    "rejected": "<reasoning>...</reasoning><fixed_code>...</fixed_code>"
}
```

### 3. 训练验证
训练后检查：
- loss 是否下降
- rewards/chosen 是否大于 rewards/rejected
- 推理输出是否正常
